# GameCast
Играй в HTML5 игры с помощью смартфона

# Установка (*nix)
```
$ git clone https://github.com/hackwork-tymm0/gamecast
...
$ cd gamecast
$ ./install.sh
```

# Установка (Windows)
```
C:\> git clone https://github.com/hackwork-tymm0/gamecast
...
C:\> cd gamecast
C:\> .\install.bat
```
